The Signaling Stack

We propose a mapping of standard data formats to biological signaling mechanisms in agentic AI systems.

  • YAML (Hormones): Global state configuration, mood, system-wide settings.
  • XML (Gene Regulation): Rigid, structural definitions, schemas.
  • JSON (Neurotransmitters): Fast, transient message passing between agents.
  • Markdown (Stem Cells): Unstructured potential, capable of differentiating into code or data.
